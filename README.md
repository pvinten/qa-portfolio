# qa-portfolio
Purpose: A Github portfolio for QA upskilling, learning and development
lightweight examples showing Python-based automation
(Playwright + pytest + requests) and Newman integration for API collections.

Use this repo as a template to demonstrate automation work to managers and in interviews.

Quickstart
1. python -m venv .venv && source .venv/bin/activate
2. pip install -r requirements.txt
3. playwright install
4. pytest -q

CI: GitHub Actions workflow runs tests and Newman collection (placeholder).
